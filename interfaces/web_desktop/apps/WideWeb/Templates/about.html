About Wideweb

Wideweb is a web browser for FriendUP resources. It does not read urls like http://google.com/ like you are used to, but instead reads HTML5 files on your disk drives, as well as content located on hosts in the Friend Network.

To read more about Wideweb, please refer to the documentation.

Wideweb is often times used to read documentation and help files. But it becomes very useful when accessing remote disk drives that you have connected using the Disk catalog.

Enjoy a new world of networked content using Wideweb!
